## Architecture Principles

- **No fallbacks.** If Gemma 4 / Ollama is unavailable, the system fails with a visible error.
- **No simulations.** Every attack prompt, target response, threat analysis, and verdict is generated by Gemma 4.
- **No rule-based logic.** Zero hardcoded templates, heuristics, or scoring formulas.
- **Full traceability.** Every agent response carries `model`, `source`, `latency_ms`, `raw_output`.
